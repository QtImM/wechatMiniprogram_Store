- **模块化设计**：采用 Spring Boot Starter 模式，将功能拆分为 `shop-common`（通用 POJO/异常）、`shop-starter-web`（Web 配置）、`shop-starter-mybatis`（持久层增强）和 `shop-starter-security`（安全认证），通过 `META-INF/spring/org.springframework.boot.autoconfigure.AutoConfiguration.imports` 实现自动装配。
- **分层边界**：`shop-common` 定义跨模块契约（如 `CommonResult`、`PageResult`）；`shop-starter-mybatis` 提供 `BaseDO` 和 `BaseMapperX` 以标准化数据访问层；`shop-starter-security` 集成 Spring Security 与自定义 `TokenAuthenticationFilter`，实现基于 Redis 的 Token 校验。
- **依赖方向**：各 Starter 模块均依赖 `shop-common` 以确保异常处理和响应格式的一致性，形成以 `shop-common` 为核心的底层支撑架构。

- **框架核心**：采用 `uni-app` (Vue 3 + TypeScript) 实现跨端开发，通过 `vite-plugin-uni` 进行构建，目标平台为微信小程序 (`mp-weixin`)。
- **状态管理**：使用 `Pinia` 作为全局状态管理库，在 `src/main.ts` 中初始化并挂载到 Vue 实例。
- **网络层封装**：在 `src/api/request.ts` 中统一封装 `uni.request`，处理 Base URL、Token 自动注入（从 Storage 获取）、以及统一的错误提示和 401 未授权跳转逻辑。
- **模块化 API**：业务接口按领域拆分，如 `src/api/product.ts` 定义了商品与分类的 TypeScript 接口及请求方法，实现类型安全。
- **页面路由配置**：通过 `src/pages.json` 集中管理页面路径与全局导航栏样式，首页 `pages/index/index.vue` 采用 Composition API (`<script setup>`) 组织逻辑。

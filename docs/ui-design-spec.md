# UI 设计方案（参考远方好物风格）

---

## 1. 设计风格定位

| 维度 | 方案 |
|------|------|
| 风格 | 简洁大方、自然质朴、信任感强 |
| 主色 | **#5B8C5A 草木绿**（沉稳自然/有机健康/区别于远方好物的亮翠绿） |
| 主色浅 | #E8F2E7（按钮浅底/标签背景/输入框背景） |
| 主色深 | #3D6B3C（按下态/深色文字） |
| 辅色 | **#B8860B 暗金色**（会员/高端感/VIP标识） |
| 辅色浅 | #FBF4E4（会员卡片背景） |
| 强调色 | **#E07C4F 陶土橙**（促销标签/活动角标/重要提示） |
| 背景 | #F6F7F4 微暖灰绿底 + #FFFFFF 白色卡片 |
| 文字 | #2D3A2E 主文字 / #5C6B5D 副文字 / #9CA89D 辅助 |
| 价格 | #CF4A3E 暗红色（比纯红柔和，更协调） |
| 圆角 | 卡片 20rpx / 按钮 44rpx / 图标 50% |
| 字体 | 系统默认，标题 32-36rpx，正文 28rpx，辅助 24rpx |
| 阴影 | 卡片用 0 4rpx 16rpx rgba(91,140,90,0.08) 微绿色阴影 |

### 配色对比（与远方好物的区分）

| 元素 | 远方好物 | 我们 | 差异 |
|------|----------|------|------|
| 主色 | #43B883 翠绿（亮/冷） | #5B8C5A 草木绿（沉/暖） | 我们更沉稳自然 |
| 氛围 | 科技感/现代 | 质朴/有机/大地 | 更贴合农特产品 |
| 会员 | 棕金 | #B8860B 暗金 | 更低调高级 |
| 强调 | 无明显强调色 | #E07C4F 陶土橙 | 增加活动氛围感 |
| 背景 | 纯灰 #F5F5F5 | 微暖灰绿 #F6F7F4 | 整体更统一 |
| 文字 | 纯黑/灰 | 绿灰色系 | 色彩更一体 |

### 首页关键设计点

1. **顶部**：品牌Logo + 名称 + slogan，建立品牌认知
2. **搜索栏**：胶囊形圆角，浅绿底色，左侧搜索图标
3. **分类Tab**：横向滑动，选中态加粗+下划线+主色
4. **Banner**：大圆角(16rpx)，支持跳转商品/活动/外链
5. **金刚区**：5个圆形图标入口，图标下方文字，一行展示
6. **双栏区**：参考远方好物的左右分栏布局，信息密度高
7. **商品流**：大图+文字模式为主，高转化
